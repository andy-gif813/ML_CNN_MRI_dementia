# CNN-based MRI Dementia Classification

This project uses Convolutional Neural Networks (CNN) to classify dementia status from MRI brain images.

The model is designed to automatically extract imaging features and predict dementia risk, supporting early diagnosis and clinical decision-making.

## Dataset includes:
Three types of MRI images.
- No impairment
- Mild impairment
- Moderate impairment

## Model Architecture
The CNN model includes:
- Convolutional layers
- Max pooling layers
- Flatten layer
- Fully connected layers
